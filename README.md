# Snake Game

## How to download
```bash=
git clone https://github.com/RickCompileError/SnakeGame.git
```

## dependency
```bash=
sudo apt install make gcc
sudo apt-get install libncurses5-dev libncursesw5-dev
```

## How to start
```bash=
git checkout network-snake
make
make active_client
```

## how to clena relative file
```bash=
make clean
```
